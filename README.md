## Steps to run locally: 

1. Git Clone `git clone https://github.com/parikrut/React-JSON`.
2. CD into folder and run
    ```javascript
        npm install
        npm start
    ```
3. Visit `http://localhost:3000/`

## Dependencies

- Node 12.x or above
- NPM 5.x or above
